# E-COMMERCE-SALES-Analysis
Exploratory Data Analysis of "Superstore Dataset" to uncover Sales Trends, Profitability Drivers, and Business insights.

## 📌 Project Overview:

- This project analyzes a retail superstore dataset to uncover business insights related to sales trends, profitability, customer behavior, discount impact, and shipping performance.

## 📊 Dataset Information:

- 9,994 rows
- 21 columns
- No missing values
- No duplicates

## 🔎 Key Analysis Performed:

- Year-wise Sales & Profit Trends
- Monthly Performance Analysis
- Category & Sub-category Profitability
- Regional Performance Analysis
- Discount vs Profit Relationship
- Customer Behavior Analysis
- Shipping Performance Analysis

## 💡 Key Insights:

- Sales and profit show consistent year-over-year growth.
- Technology category generates the highest profit.
- Heavy discounting negatively impacts profitability.
- Some sub-categories generate losses.
- Customer retention can be improved.
- Shipping efficiency varies across modes.

## 🛠 Tools Used:

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 🚀 Conclusion:

- The business is profitable and growing, but optimizing discount strategy and focusing on high-margin categories can further improve performance.
